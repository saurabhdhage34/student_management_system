# EduManage Pro

# Project Description

**EduManage Pro** is a powerful Student Management System developed using Django.
It helps manage students, courses, attendance, exams, and fees efficiently in one platform.

---

# Features

*  Student Management (Add / Update / Delete)
*  Fees Management
*  Tasks / Assignments
*  Student Dashboard
*  Exam Management
*  Notices / Announcements
*  Courses Management
*  Batches Management
*  Reports
*  Attendance System

---

#  Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite
